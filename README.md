# Yue Guo — PM & TPgM Portfolio

**Live site:** https://yueg18.github.io/Yue-Guo-Portfolio-Website

---

## Overview

Personal portfolio website for Yue Guo, a Product Manager and Technical Program Manager with 9+ years of experience across fintech, insurance, and enterprise platform delivery.

Built as a single-page application with tabbed navigation — no frameworks, no dependencies, pure HTML/CSS/JS.

## Sections

| Tab | Content |
|-----|---------|
| **About** | Background, strengths, testimonials, and CTA |
| **Superpowers** | Three core differentiators with real outcome metrics |
| **Featured Works** | 16 case studies across Fintech, Insurance, Consulting, Analytics, and Financial Systems — each with an interactive data diagram |
| **Tech Stack & Methodology** | Skill tags with hover context, plus a 3-step decision framework |
| **Contact** | Two-column layout with intro and a live contact form (Formspree) |

## Tech

- Vanilla HTML / CSS / JS — zero build step
- Google Fonts: Playfair Display + DM Sans
- Form handling: [Formspree](https://formspree.io)
- Deployed via GitHub Pages

## Updating the site

Edit `index.html` locally, then push:

```bash
git add index.html
git commit -m "your update message"
git push
```

GitHub Pages will redeploy automatically within ~1 minute.
